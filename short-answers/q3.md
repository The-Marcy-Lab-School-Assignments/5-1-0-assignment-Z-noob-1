# Question
In a code snippet, demonstrate a `User` class with at least one:
- class property
- class method
- instance property
- instance method

# Answer
```js
class User {
  static #allUsers = [];

  constructor(id, accounts) {
    this.id = id;
    this.accounts = accounts; // Array 
    User.#allUsers.push(this); 
  };

  getAccounts() {
    return [...this.accounts];
  };

  static getUserInfo() {
    return User.#allUsers;
  };
}
```
